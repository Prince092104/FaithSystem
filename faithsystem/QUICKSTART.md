# Quick Start Guide - Club Membership Manager

## 5-Minute Setup

### Step 1: Create Database (2 minutes)
1. Open XAMPP and start Apache and MySQL
2. Go to http://localhost/phpmyadmin
3. Click **SQL** tab
4. Copy entire content from **database.sql** file
5. Paste and click **Go**

### Step 2: Start Using the System (1 minute)
1. Open http://localhost/faithsystem/
2. You're ready to use the system!

---

## First Time Usage

### Create Your First Member
1. Click **Register Member** button
2. Fill in the form:
   - First Name: John
   - Last Name: Doe
   - Email: john@example.com
   - Phone: 555-1234
   - Date of Birth: 1990-01-15
   - Join Date: Today's date
   - Membership Type: Regular
3. Click **Register Member**

### View Members
1. Click **Members** in menu
2. See all registered members
3. Use search to find specific members

### Generate Reports
1. Click **Reports**
2. Choose Summary or Detailed
3. Download as CSV or PDF

## Membership Status Types

- **Active** - Member is current and active
- **Inactive** - Member not participating
- **Suspended** - Temporary suspension
- **Expired** - Membership expired

## Common Tasks

### Find a Member
- Go to Members → Search box → Type name/email → Enter

### Change Member Status
- Click member name → Edit button → Change status → Save

### Generate Report
- Reports → Choose type → Click export format

### Backup Data
- Reports → Export to CSV → Save file

## Troubleshooting

**Issue**: Can't connect to database
- **Solution**: Check MySQL is running in XAMPP

**Issue**: Page shows error
- **Solution**: Clear browser cache (Ctrl+F5)

**Issue**: Missing data after server restart
- **Solution**: Data is saved in database, reload page
